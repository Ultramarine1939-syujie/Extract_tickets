# 解析规则说明

当前解析器面向 12306 / 国铁电子客票文本型 PDF。图片型或扫描件 PDF 不会进行 OCR，会返回“图片型PDF，需OCR，暂跳过”。

## 字段来源

- `发票号码`：匹配 `发票号码：` 或 `发票号码:` 后的数字。
- `开票日期`：匹配 `YYYY年M月D日`。
- `出发站 / 车次 / 到达站`：匹配单行 `出发站 车次 到达站`，车次为字母加 3 到 5 位数字。
- `乘车日期 / 开车时间 / 车厢号 / 座位号 / 席别`：匹配 `YYYY年M月D日 HH:MM开 NN车座位号 席别`，兼容卧铺的上铺/中铺/下铺。
- `票价(元)`：匹配 `¥` 或 `￥` 后的金额。
- `证件号(脱敏) / 乘车人`：匹配 `6位数字 + **** + 4到6位数字 + 中文姓名`。
- `电子客票号`：匹配 `电子客票号：` 或 `电子客票号:` 后的数字。

## 维护建议

新增票据版式时，先将真实 PDF 文本脱敏后抽成字符串样本，再补测试。不要提交真实票据原件。
